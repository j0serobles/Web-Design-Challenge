Climate vs. Latitude

This page provides the visualizations created for the results of our weather analysis for 600 cities, comparing the temperature, wind speed, cloudiness and humidity against the city's geographical latitute.

The weather data and cities listing were retrieved from the API at openweathermap.org.

The process followed to create the weather dataset was:

  • Download the cities listing (in json format) from the website and upload it into a Pandas dataframe.
  • Randomly select 600 cities from the full list, and obtain the weather facts (temperature, wind speed, cloudiness and humidity) for each one, the data was then uploaded to another Pandas dataframe and merged with the cities dataframe. The weather data was retrieved using the API on March 13, 2021 at 7:20PM.
We then used the matplotlib python library to produce the visualizations, using scatter plots to compare:
  • Temperatures against latitue (Visualization 1)
  • Humidity against latitute (Visualization 2)
  • Cloudiness against latitute (Visualization 3)
  • Wind speed against latitude (Visualization 4)

Click on the thumbnail images on the right to access the visualization's page. You can also access them via the "Plots" menu on the navigation menu. Two additional pages are provided with additional information:

  • Comparisons - Displays all scatter plots in one page, for comparison
  • Data - Shows the full table with the data that was used to produce the visualizations
